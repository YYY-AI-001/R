# 源力魔方副业系统 PRD

版本：v0.1  
日期：2026-05-25  
形式：HTML 展示页

## 文件说明

- `index.html`：源力魔方副业系统 PRD 的 HTML 展示版。

这份 PRD 用于梳理「源力魔方 / SourceCube」当前副业工作内容，包括：

- B站、YouTube、公众号、知识星球内容体系；
- RunningHub 工作流推广；
- 校园公开课；
- 3 天 AIGC 提升营；
- HTML 汇报页 / 汇报 PPT 工具；
- 后续团队与资源协作判断。

## 当前判断

项目方向成立，但仍处于早期验证阶段。

当前不急着成立工作室、不急着拉合伙人、不急着搭复杂平台。优先目标是通过内容、公开课和 3 天提升营 MVP 验证真实需求。

## 使用方式

本地打开：

```text
prd/sourcecube-side-business-prd/index.html
```

浏览器打开后可作为 PRD 汇报页、项目复盘页或继续迭代的产品文档使用。
